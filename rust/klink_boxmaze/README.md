# klink-boxmaze-rs

Internal Rust acceleration kernel for
[klayout-klink](https://github.com/klinkdev2026/klayout-klink) — a byte-parity
port of klink's pure-Python detailed-routing reference, shipped as pre-built
wheels. It changes speed only, never results.

Do not install this package directly; install `klayout-klink`, which depends
on the matching kernel version. Apache-2.0.
